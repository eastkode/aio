About STD and ISD Codes

Subscriber Trunk Dialing (STD) codes are used for making long-distance calls within a country, while International Subscriber Dialing (ISD) codes are used for making international calls.

Frequently Asked Questions (FAQs)

What is an STD code?
An STD code is a unique number assigned to a specific city or region within a country, used for making long-distance calls.
What is an ISD code?
An ISD code is a unique number assigned to each country, used for making international calls.
How can I find an STD or ISD code?
You can use our STD/ISD Code Lookup tool to find the code for any city or country.
