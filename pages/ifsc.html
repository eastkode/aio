About IFSC Codes

The Indian Financial System Code (IFSC) is an 11-character code assigned by the Reserve Bank of India (RBI) to identify every bank branch participating in the payment system. The first four characters represent the bank, the fifth is a '0', and the last six characters represent the branch.

Frequently Asked Questions (FAQs)

What is an IFSC code?
An IFSC code is an 11-character alphanumeric code that uniquely identifies a bank branch in India.
Why is an IFSC code required?
It is used for electronic fund transfers such as RTGS, NEFT, and IMPS.
How can I find an IFSC code?
